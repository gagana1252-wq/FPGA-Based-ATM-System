# FPGA Based ATM System using Verilog
### Mini Project (BEC613D) | Department of ECE, SJBIT | 2024-25

---

## 🧩 Project Overview
This project demonstrates the design and simulation of an **Automated Teller Machine (ATM)** system using **Verilog HDL**.  
The system handles core functionalities such as **PIN verification, balance checking, withdrawal, and deposit operations**.  
It is implemented and tested on an **FPGA board** for real-time operation.

---

## 🏗️ Features
- Secure PIN verification
- Balance management
- Deposit and withdrawal transactions
- Testbench for simulation
- Real-time FPGA implementation with switches and LEDs

---

## 🧠 New Skills Acquired
- Verilog HDL Programming  
- FPGA Board Design & Pin Mapping  
- Simulation & Testbench Debugging  
- Sequential Logic Design & State Machines  
- Embedded System Design for Banking Applications

---

- **Language:** Verilog HDL
